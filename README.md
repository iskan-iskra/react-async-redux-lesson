# React Async Redux lesson:

## Версии:

- node: 20.11.1
- npm: 10.2.4

## Команды:

### установить зависимости:

```
npm install
```

### запустить локально проект:

```
npm run dev
```

### собрать билд проекта:

```
npm run build
```

## Стек:

- Vite
- JavaScript
- React
- React-Router-DOM
- React-bootstrap
- Redux
- React-Redux
- Redux-Thunk
- Redux-Saga
